Массивы

список

упорядоченная последовательность элементов

["string", 1, ["list", variable]]
Порядок в списке имеет значение

a = [1, 2, 3]
b = [1, 2, 3]
c = [3, 2, 1]
print(a == b) # True
print(a == c) # False

Характеристика массивов

Динамическая длина

не имеют фиксированной длины

Гетерогенность

в списке могут находиться элементы различных типов данных

Индексация

первый элемент - 0, однако поддерживается и обратная индексация, где -1 - последний элемент списка

Изменяемость

их содержимое может изменяться после создания

Характеристика массивов

Поддерживают срезы

можно извлекать подсписки с помощью оператора среза


my_list = [1, 2, 3, 4, 5]
print(my_list[1:4])  # [2, 3, 4]

Вложенность

списки могут содержать другие списки 


matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print(matrix[0][1])  # 2 (элемент первой строки и второго столбца)

позволяет перебирать их элементы в цикле for


my_list = ["apple", "banana", "cherry"]
for fruit in my_list:

Операции с несколькими списками

Списки можно складывать (конкатенация) и умножать на число


list1 = [1, 2]

print(list1 + list2)  # [1, 2, 3, 4]
print(list1 * 3)  # [1, 2, 1, 2, 1, 2]

Характеристика массивов

Копирование списков

присваивание одного списка другому передает ссылку на один и тот же объект

my_list = [1, 2, 3]
any_list = my_list
any_list.append(4)
print(my_list) # [1, 2, 3, 4]
print(any_list) # [1, 2, 3, 4]

copy_list = my_list[:]  # Создаем копию с помощью среза
copy_list.append(5)
print(my_list) # [1, 2, 3, 4]
print(copy_list) # [1, 2, 3, 4, 5]

Вывод списка

Одновременный вывод индексов и элементов

можно использовать функцию enumerate()


my_list = ["яблоко", "банан", "вишня"]
for index, item in enumerate(my_list):
    print(f"Индекс {index}: {item}")

# Индекс 0: яблоко
# Индекс 1: банан
# Индекс 2: вишня

Вывод списка

Вывод элементов через разделитель

для вывода элементов списка в одной строке


my_list = ["яблоко", "банан", "вишня"]
print(" ".join(my_list))  # "яблоко банан вишня"
join() работает только со строками

Методы работы со списками

append()

добавляет элемент в конец списка


my_list = [1, 2, 3]
my_list.append(4)
print(my_list)  # [1, 2, 3, 4]
extend()

добавляет в его конец элементы из другого списка


my_list = [1, 2, 3]
my_list.extend([4, 5])
print(my_list)  # [1, 2, 3, 4, 5]

Методы работы со списками

insert()

вставляет элемент на определенную позицию в списке


my_list = [1, 3, 4]
my_list.insert(1, 2)  # Вставляем 2 на индекс 1
print(my_list)  # [1, 2, 3, 4]

1 - позиция, 2 - элемент

remove()

удаляет первое вхождение указанного элемента в списке


my_list = [1, 2, 3, 2]
my_list.remove(2)  # Удаляет первое вхождение 2
print(my_list)  # [1, 3, 2]
Если элемент не найден, возникает ошибка ValueError

Методы работы со списками

pop()

удаляет элемент по указанному индексу и возвращает его


my_list = [1, 2, 3]
last_item = my_list.pop()  # Удаляем последний элемент
print(last_item)  # 3
print(my_list)  # [1, 2]
index()

возвращает индекс первого вхождения указанного элемента


my_list = [1, 2, 3, 2]
print(my_list.index(2))  # 1 (индекс первого вхождения 2)
Если элемент не найден, возникает ошибка ValueError

Методы работы со списками

count()

возвращает количество вхождений указанного элемента в списке


my_list = [1, 2, 3, 2]
print(my_list.count(2))  # 2
reverse()

меняет порядок элементов в списке на обратный


my_list = [1, 2, 3]
my_list.reverse()
print(my_list)  # [3, 2, 1]

Методы работы со списками

sort()

сортирует список на месте (изменяет исходный список), упорядочивая элементы по возрастанию


my_list = [3, 1, 4, 1, 5]
my_list.sort()
print(my_list)  # [1, 1, 3, 4, 5]
Для обратной сортировки используется параметр reverse=True

my_list.sort(reverse=True)
print(my_list)  # [5, 4, 3, 1, 1]

Методы работы со списками

copy()

создает поверхностную копию списка


my_list = [1, 2, [3, 4]]
new_list = my_list.copy()
new_list += [5]
print(my_list)  # [1, 2, [3, 4]]
print(new_list)  # [1, 2, [3, 4], 5]

my_list[2][0] = 0
print(my_list)  # [1, 2, [0, 4]]
print(new_list)  # [1, 2, [0, 4], 5]

Арифметические операции в списках


ratings = [2.5, 5.0, 4.3, 3.7]

print(mix(ratings)) # 2.5
print(max(ratings)) # 5.0
print(sum(ratings)) # 5.0
